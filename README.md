# How to clone a repository
1. Open your terminal and navigate to the directory where you want to clone the repository.
2. Run the following command, replacing <repository-url> with the URL you copied:
   a) git clone <repository-url>
3. Once the cloning process is complete, navigate into the repository folder:
   a) cd <repository-name>
